# Frontend/docs/

- `openapi_main.yaml` は親リポジトリ（`docs/openapi_main.yaml`）から同期されるファイル。絶対に編集しないこと
- `component_design.md` と `routing.md` はフロントエンド実装の設計ドキュメント。実装変更時は対応する箇所を更新すること
- `technical_policies.md` はフロントエンド固有の技術方針。技術スタック・認証方式・API クライアント方式等を変更した場合は更新すること
- 影響範囲: `component_design.md` や `routing.md` を変更した場合、対応する実装（`pages/`, `components/`, `router/`）との整合性を確認すること
