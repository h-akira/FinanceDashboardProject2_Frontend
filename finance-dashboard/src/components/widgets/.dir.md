# Frontend/finance-dashboard/src/components/widgets/

- 各ウィジェットは props/emits を持たない自己完結型コンポーネント
- 新規ウィジェット追加時は以下も更新すること:
  1. `pages/DashboardPage.vue` の `availableCharts` に `ChartEntry` を追加
  2. `Frontend/docs/component_design.md` のウィジェット一覧を更新
- 外部スクリプト埋め込みは `onMounted` で読み込み、`onBeforeUnmount` でクリーンアップする
- API データの可視化には Lightweight Charts を使用する（Chart.js 等は不可）
- チャートは `onBeforeUnmount` で `chart.remove()` を実行し、リサイズ対応すること
