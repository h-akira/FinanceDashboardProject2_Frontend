# Frontend/finance-dashboard/src/pages/

- コンポーネント設計は `Frontend/docs/component_design.md` を参照
- ルーティングは `Frontend/docs/routing.md` を参照
- 新規ページ追加時は以下も更新すること:
  1. `router/index.ts` にルート定義を追加
  2. 認証が必要なら `meta: { requiresAuth: true }` を設定
  3. 利用頻度が低いページは遅延読み込み `() => import(...)` にする
  4. 必要に応じて `components/AppHeader.vue` にナビゲーションリンクを追加
  5. `Frontend/docs/component_design.md` と `Frontend/docs/routing.md` を更新
