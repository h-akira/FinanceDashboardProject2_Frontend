# Frontend/finance-dashboard/src/auth/

- 認証方式は Cognito Managed Login + PKCE
- トークンは sessionStorage に保存する（localStorage 不可）
- トークン読み取りは `getIdToken()`, `getAccessToken()` 経由で行う
- `import.meta.env.DEV` 時は Cognito 通信を行わずモックトークンを使用する
- 影響範囲: `api/custom-fetch.ts`（トークン取得）、`pages/CallbackPage.vue`（トークン交換）、`pages/ChangePasswordPage.vue`（Access トークン）、`components/AppHeader.vue`（認証状態）
- 認証フローを変更した場合は `Frontend/docs/technical_policies.md` の認証方針セクションも更新すること
