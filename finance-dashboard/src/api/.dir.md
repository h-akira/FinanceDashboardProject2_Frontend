# Frontend/finance-dashboard/src/api/

- `generated/` は Orval 自動生成。手動編集しない
- 全 API リクエストは `custom-fetch.ts` の `mainFetch` 経由で行う
- `custom-fetch.ts` が Authorization ヘッダー付与・401 リトライ・強制ログアウトを担う
- API 定義の変更時は `npm run generate:api` で再生成する
- 影響範囲: `custom-fetch.ts` を変更すると全 API 呼び出し（全ページ・ウィジェット）に影響する
- API クライアントの方式を変更した場合は `Frontend/docs/technical_policies.md` の API クライアント方針セクションも更新すること
